# EC2 Apache Web Server

This project demonstrates hosting a static website on an AWS EC2 instance using Apache (httpd).

## What I did
- Launched an EC2 instance (Amazon Linux 2023)
- Configured Security Groups (HTTP & SSH)
- Installed and started Apache web server
- Deployed a custom HTML page
- Accessed the site via public IPv4 address

## Tech Stack
- AWS EC2
- Amazon Linux 2023
- Apache HTTP Server
- Linux CLI

## Output
The webpage is accessible via the EC2 public IP.
